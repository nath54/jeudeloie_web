Comment jouer ?

C'est une version remasterisée non officiele du jeu de plateau de l'oie, donc elle se joue quasiment pareil, c'est a dire que les joueurs lancent chacun un dé pour faire avancer leurs oies

But du jeu :

La première oie arrivant à la case 63 a gagnée

Règles :

Voici une liste des règles originelles du jeu de l'oie :

  • Les joueurs jouent les uns après les autres
  • Les joueurs avancent leurs pions du meme nombre de case que le résultat du lancer de dé
  • Quand les joueurs dépassent 63, ils rebondissent et retournent en arriere

Voici une liste de règles personnelles, ou présentes dans le jeu de l'oie, mais modifiées :

  • Quand un joueur va sur la case 42, il a le droit de relancer les dés
  • Quand un joueur va sur la case 13, il est immobilisé pendant 2 tours
  • Quand un joueur va sur une case multiple de 10, il reculera la prochaine fois qu'il lancera les dés
  • Quand un joueur va sur la case 62, il retourne a la case départ
